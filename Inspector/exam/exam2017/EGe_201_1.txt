The graphs show us information about using Facebook from 2012 to 2013 year. 
The first chart is about number of users and diffeneces between using Facebook on desktop and mobile. The number of users of Facebook on desktop falls from 150 millions on March, 2012 to about 130 millions people on March 2013. At the same time the number of using Facebook on phones is increasing : from 50 millions on March, 2012 to almost 100 millions users on March, 2013. It can be seen that using Facebook on phone become more popular, while desktop version is losing its users. However, generally more people use Facebook. 
The second graph shows us information on reasons to use Facebook. The main reason among people is sharing photos and videos, it is 54% for women and 39% for men. The same popularity for men has receiving updates, it ts also 39%. Intesresting thing that it has 39% for women too. However, the second reason is different for women - it is seeing funny posts, which has 43%. For men this reason is the main only for 35% of them. Meanwhile, the less popular reason for both sexes is learning about ways to help others. It has 35% among women and 25% among men.
